Font Options

  • Plus Jakarta Sans
  • Playfair Display

Plus Jakarta Sans is the default font for Heading & body in NextAI. If you wish to use one of the other available font options, add the link of the font CSS file you wish to use inside the head tag, right before style.css link, in all your HTML pages.

Or, The font scss file can be separeted from page to page and import in src/assets/scss/fonts/ff-1.scss
